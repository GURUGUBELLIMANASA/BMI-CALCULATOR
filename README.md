# BMI Calculator 🧮

A simple command-line BMI (Body Mass Index) Calculator built using Python.

This project takes the user's weight (in kilograms) and height (in meters), calculates BMI using the standard formula, and displays the BMI value along with the weight category.

## Features

- Takes user input for weight and height
- Calculates Body Mass Index (BMI)
- Displays BMI result
- Classifies BMI into categories:
  - Underweight
  - Normal weight
  - Overweight
  - Obesity

## BMI Formula

BMI = weight (kg) / height² (m²)
